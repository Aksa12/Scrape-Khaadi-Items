# Scrape-Khaadi-Items
Python Script that scrapes new items information from Khaadi.

It uses Selenium and Beautiful Soup to extract Items. The code extracts items information from NEW IN section of Khaadi's website, i.e., https://pk.khaadi.com/new-in.html.
The information, product name, price, link, available colors, is saved in the clothes.csv file.
